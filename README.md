# Weather Dashboard

A simple Weather Dashboard built with **React** and **Tailwind CSS**.  
This project allows users to search for a city and view its weather information (to be connected with a weather API).

---

## Features

- Search for any city.
- Responsive design using Tailwind CSS.
- Input validation (alerts user if the search field is empty).
- Fully centered layout for better user experience.
